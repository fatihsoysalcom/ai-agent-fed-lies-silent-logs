# AI Agent Fed Lies Silent Logs

This example demonstrates how an AI agent can be fed manipulated data, leading to an incorrect decision. It highlights how traditional logging, which records the data processed by the AI, might not reveal the underlying data manipulation, making it difficult to detect 'lies' fed to the agent.

## Language

`python`

## How to Run

Save the code as `main.py`.
Run from your terminal: `python main.py`
